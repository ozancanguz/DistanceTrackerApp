# Distance Tracker App

I have been developing a distance tracker android mobile app.
Still on progress... 

**Tools that I used** :

- Navigation Component
- Hilt for dependency injection
- Android Services
- Google Maps SDK
- Coroutines
- Livedata
- Data Binding
- View Binding
- Fused Locations
- Easy Permissions 



**Arhictecture:**

- MVVM
